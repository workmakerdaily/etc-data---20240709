Node.js 및 npm(node package manager)이 설치되어 있어야 함.

1.  Create React App 설치: 터미널에서 다음 명령어를 실행한다.
# JavaScript
npx create-react-app project-name

# TypeScript
npx create-react-app project-name --template typescript
여기서 'project-name'은 생성하려는 앱의 이름이다. 원하는 이름으로 변경 가능하다.

2. 설치된 애플리케이션으로 이동
cd project-name

3. 애플리케이션 시작
npm start
